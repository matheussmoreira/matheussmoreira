
# HELLO, I'M MATHEUS! 👋

### About me
👨🏻‍💻 Low-income student, I'm studying Computer Science at PUC-Rio as a scholarship holder.

🍎 I am a former student of the Apple Developer Academy PUC-Rio, where I participated in 2020-21.
